# pydino
PyDino is a robot to play Google Dino Game.

### [tested on]:
```
python 3.7.3
```
Windows 10 1909
```
Chrome 80.0.3987.163
```
Resolution: 1366x768
```

### [to install]:
```
pip install -r requeriments.txt
```

### [to run]:
```
Open the site chrome://dino in our Google Chrome
```
Execute: python pydino.py
```
